Job * parse ( char *  line)

Parse the command line.

This function parses the input command line and generates a corresponding job if it's valid. It uses a finite state machine to parse commands and special grammer token such as '|' for piping and '&' for background.

Parameters
lineString of the input command line read from readline.
Returns
Pointer to the generated job. NULL if the input is illegal.
