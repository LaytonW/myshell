int launchJob ( Job *  j)

Launch a job .

This function takes a job and launches it. It handles background/foreground setting, pipe setting, builtin function looking up before invoking the process(es). It then calls launchProcess() to launch the process(es) and waits for foreground process(es) to terminate. Then it performs cleaning operations.

Parameters
jPointer to the job to be launched.

Exit status.
