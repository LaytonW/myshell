int launchJob ( Job *  j)

Launch a job .

This function takes a job and launches it. It handles background/foreground setting, pipe setting, builtin function looking up before invoking the process(es). It then calls launchProcess() to launch the process(es) and waits for foreground process(es) to terminate. Then it performs cleaning operations.

void launchProcess ( Process *  p,
pid_t  pgid,
int  in,
int  out 
)

Launch a process .

This function is called in child processes after fork(). It redirects input/output of process p for piping to in and out if they are not standard input/output. It then sets the process id of p to pgid for background/foreground management. After receiving the signal SIGUSR1, it calls execvp() to invoke the command of the process.

Parameters
pPointer to the process to be launched.
pgidProcess group id to be set to the process.
inInput file descriptor.
outOutput file descriptor.
